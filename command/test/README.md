# test

Test files types and details.

```fish
if test -d my_folder
	echo "Is folder"
end
```


## Options

- File is not empty (size not 0) with `-s`

- Folder with `-d`

- Path exists with `-e`

- Regular file with `-f`

- Symbolic link with `-L`


### Permissions

- Executable file with `-w`

- Readable file with `-r`

- Writable file with `-w`
