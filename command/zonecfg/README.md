# zonecfg

## Subcommands

Multiple subcommands can be separated with semicolons (;).

- [add](./add/)

- cancel

- clear

- commit

- create

- delete

- end

- exit

- [export](./export/) to print the configuration for a command file.

- help

- [info](./info/) to print the current configuration.

- remove

- revert

- select

- [set](./set/)

- verify
